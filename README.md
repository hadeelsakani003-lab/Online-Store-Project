Online Store Management System
Description
An interactive Java console application designed to manage an online store's operations. The system handles product inventory, customer registrations, and a dynamic shopping cart system, culminating in a detailed checkout process.

Technologies & Tools
Language: Java.

Paradigm: Object-Oriented Programming (OOP).

Development Environment: Apache NetBeans.

Version Control: Git & GitHub.

Project Responsibilities (Key Features)
Inventory Control: Implementing a robust system to add and track products (Books, Games, Shoes) with real-time stock management.

Customer Management: Creating and managing customer profiles with the ability to update IDs and names.

Shopping Cart Logic: Developed logic for adding/removing items to a personal cart, ensuring quantities are automatically synchronized with the store's stock.

Billing & Checkout: Implementing an automated billing system that calculates sub-totals, totals, and generates a final invoice for the customer.

Core OOP Concepts Applied
Inheritance: Used to create a hierarchy between the base Item class and specialized types like BookItem.

Encapsulation: Ensuring data integrity by using private fields and public getters/setters.

Polymorphism: Method overriding to provide custom descriptions for different item categories.

Association/Composition: Managing the relationship between Customer, ShoppingCart, and Item objects.
